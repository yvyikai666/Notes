## python综合应用题

### 1. 选出一篇文章中最多的字符

```python
with open('命运.txt', 'r') as f:
    txt = f.read()
d = {}
for i in txt:
    if i not in '，。？！《》【】’“：':
        d[i] = d.get(i, 0) + 1  # 先在d中查找，若没有就返回0

ls = list(d.items())  # 将一个字典转换为一个列表，其中每一个键值对都变成一个元组
ls.sort(key=lambda x: x[1], reverse=True) # 按照值的大小preverse=True说明是倒序排列(y)
print("{}:{}".format(ls[0][0], ls[0][1]))
```
